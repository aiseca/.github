# Contributing to AISECA

Thank you for your interest in contributing to the AI Security Alliance.

## How to Contribute

### Framework Feedback
- Open an issue in the [`framework`](https://github.com/aiseca/framework) repo with the `feedback` label
- Be specific about which tier or control your feedback relates to
- Include real-world context where possible -- we value practitioner experience

### Controls Catalog
- Suggest new controls or refinements via pull requests to [`controls-catalog`](https://github.com/aiseca/controls-catalog)
- Each control should include: name, description, tier mapping, and implementation guidance

### General Discussion
- Use [GitHub Discussions](https://github.com/orgs/aiseca/discussions) for broader topics
- Be respectful and constructive

## Code of Conduct

All contributors are expected to be professional, respectful, and constructive. We are building a vendor-agnostic standard -- contributions should not promote specific commercial products.

## Membership

AISECA board membership is by application. Visit [aiseca.org](https://aiseca.org) to apply.
